# Project: Auth-Xinsi (Cloudflare Workers QQ OAuth2.0 Center)
# Context: 
- A serverless authentication middleware for lz-0315.com (辛巳学习网).
- Frontend is a static site; Backend is Cloudflare Workers.
- Purpose: Handle QQ Login, exchange tokens, and fetch user info.
